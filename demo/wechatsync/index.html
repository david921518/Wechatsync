如何安装

下载安装包

点击 下载链接 并保存文件；

打开所在的文件夹，找到下载好的 WechatSync.zip 文件解压。

打开浏览器扩展管理页面

复制 chrome://extensions ，并粘贴到地址栏，按回车键进入 扩展中心 页面。

安装插件

在扩展中心打开右上角的【开发者模式】按钮，然后刷新页面，把解压后的 WechatSync 文件夹拖入扩展中心。

还是不会？请看安装视频教程。

怎样使用

多平台互同步

任意网页右键菜单 点击插件提供的提取正文选项，可以做到从知乎同步到头条，头条同步到公众号，任意平台互同步

1. 任意网页右键菜单；

2. 点击 提取正文 选项；

3. 即可提取到当前页面到文章；

4. 选择需要同步到渠道，即可做到互同步。

从公众号同步

1. 网页打开 任意一篇公众号文章；

2. 左上角可看到 同步该文章 按钮；

3. 点击后选择要同步到的渠道账号；

4. 等待草稿同步完成，即可逐个一一发布。

同步到公众号

通过任意网页提取正文，你可以从知乎或头条、36kr、虎嗅，个人博客等任意平台、同步到公众号。

Markdown同步

从在线Markdown编辑器同步到知乎、头条等任意发布渠道。
